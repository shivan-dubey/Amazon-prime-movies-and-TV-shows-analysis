# 📊 Amazon Prime Data Analysis (Power BI Project)

## 📌 Project Overview
This project is a Power BI dashboard created to analyze Amazon Prime content data. It provides insights into movies and TV shows available on the platform.

## 🎯 Objectives
- Analyze content distribution (Movies vs TV Shows)
- Identify trends by year, genre, and country
- Understand ratings and audience categories
- Visualize key insights using interactive dashboards

## 🛠 Tools Used
- Power BI
- Data Cleaning & Transformation
- Data Visualization

## 📊 Key Insights
- Number of Movies vs TV Shows
- Content added over the years
- Top genres available on Amazon Prime
- Country-wise content distribution
- Ratings analysis

## 📁 File Included
- `AMAZON_PRIME.pbix` → Power BI Dashboard File

## 🚀 How to Use
1. Download the `.pbix` file
2. Open it using Power BI Desktop
3. Explore the interactive dashboard
